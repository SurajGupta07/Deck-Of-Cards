#### Get started

1. **Clone the repo:**
```bash
  $ git clone https://github.com/SurajGupta07/Deck-Of-Cards.git
```
2. **On `master` branch:**
```bash
  $ cd Deck-Of-Cards
```
3. **Install required node modules:**
```bash
  $ npm install
```
4. **Start the dev server:**
```bash
  $ npm start
```
Open the `localhost` link in the browser.
Happy hacking!
